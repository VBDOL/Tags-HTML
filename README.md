# 🚀 Curso de HTML - Blog Didático

Este projeto foi desenvolvido como parte do desafio proposto pela [DIO](https://web.dio.me/) para criar uma página web utilizando diversas **tags HTML** aprendidas durante as aulas.

## 📌 Objetivo

O objetivo deste projeto é ensinar **HTML básico**, mostrando exemplos de como usar tags essenciais para estruturar um site.

## 🛠️ Tecnologias Utilizadas

- **HTML5** → Para a estrutura da página
- **CSS3** → Para a estilização e melhor experiência de usuário

## 🏗️ Estrutura do Projeto

O projeto está organizado da seguinte forma:

/curso-html/ │-- index.html # Arquivo principal │-- styles.css # Arquivo de estilos │-- README.md # Documentação do projeto


## ✏️ Conteúdo Ensinado

Nesta página, você aprenderá sobre:
- Como organizar **títulos e parágrafos** (`<h1>` até `<h6>`, `<p>`)
- Formatação de texto com (`<strong>`, `<i>`, `<u>`, `<mark>`)
- Como criar **listas** (`<ol>`, `<ul>`, `<li>`)
- Como adicionar **links** (`<a>`)
- Uso de **citações e símbolos** (`<blockquote>`, `<abbr>`, `<sub>`, `<sup>`)
- Tags adicionais como (`<font>`, `<del>`)

## 🌎 Visualizar o Projeto

Para visualizar o projeto em execução:
1. Clone este repositório
2. Abra o arquivo `index.html` em seu navegador
